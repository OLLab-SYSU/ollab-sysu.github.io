# Optimization and Learning Lab. (OLLab), Sun Yat-Sen University 

## 主页仓库使用说明

本仓库用于维护和管理实验室的 GitHub Pages 主页。

### 📝 如何修改主页内容

#### 可编辑文件
目前允许修改的页面：
- 主页: **[`_pages/about.md`](_pages/about.md)**

⚠️⚠️⚠️ 需注意，`_pages/about.md`的文件头请谨慎修改。如下样例所示，文件头即为文件开头双横线框柱的内容。

```text

---
layout: about
title: about
permalink: /

nav: false
nav_order: 1

profile:
  align: right
  image: ollab.png
  image_circular: false # 是否将图片裁剪为圆形
---

```

#### 编辑指南
1. **文件位置**：`_pages/about.md`
2. **格式要求**：使用 Markdown 格式编写
3. **内容类型**：可以添加文字、链接、图片、表格等 Markdown 支持的所有格式
4. **修改方式**：
   - 直接编辑 `_pages/about.md` 文件
   - 使用 Markdown 语法更新内容
   - 保存文件后，Git 提交并推送到 GitHub


#### 生效时间
- 修改推送到 GitHub 后，GitHub Pages 会自动重新构建网站
- 通常需要 **1-3 分钟** 网站更新才会生效
- 如果没有立即看到更改，请稍等片刻并刷新浏览器
