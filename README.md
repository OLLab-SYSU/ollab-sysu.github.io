# Sun Yat-Sen University Optimization Learning Lab

## 主页仓库使用说明

本仓库用于维护和管理实验室的 GitHub Pages 主页。

### 📝 如何修改主页内容

#### 可编辑文件
目前允许修改的页面：
- **`_pages/about.md`** - 主页内容文件

#### 编辑指南
1. **文件位置**：`_pages/about.md`
2. **格式要求**：使用 Markdown 格式编写
3. **内容类型**：可以添加文字、链接、图片、表格等 Markdown 支持的所有格式
4. **修改方式**：
   - 直接编辑 `_pages/about.md` 文件
   - 使用 Markdown 语法更新内容
   - 保存文件后，Git 提交并推送到 GitHub


#### 生效时间
- 修改推送到 GitHub 后，GitHub Pages 会自动重新构建网站
- 通常需要 **1-3 分钟** 网站更新才会生效
- 如果没有立即看到更改，请稍等片刻并刷新浏览器

### 🎨 Markdown 语法参考

一些常用的 Markdown 语法：
- **标题**：`# 一级标题`、`## 二级标题`、`### 三级标题`
- **粗体**：`**粗体文字**`
- **斜体**：`*斜体文字*`
- **链接**：`[链接文字](URL)`
- **图片**：`![图片描述](图片路径)`
- **列表**：使用 `-` 或 `1.` 开头
- **代码**：使用反引号 `` `代码` ``
